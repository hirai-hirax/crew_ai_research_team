# Open-Meteo API 設定ガイド

## 概要

天気専門エージェント用にOpen-Meteo APIツールを統合しました。これにより、APIキー不要で正確な天気情報を取得できます。

## APIキーは不要！

**Open-Meteoは完全無料で、APIキーの登録や設定が一切不要です。**

すぐに使い始めることができます。

## 機能

### 取得できる情報

- 現在の天気状況
- 7日間の予報（時間ごと）
- 気温（最高・最低）
- 湿度
- 風速
- 降水確率
- 天気コード（WMO標準）の日本語説明

### お出かけアドバイス

天気情報に基づいて自動的に以下のアドバイスを生成：
- 雨具の必要性
- 服装の推奨
- 熱中症・防寒対策
- 屋内/屋外活動の適性

## 使用例

```python
from tools.openweather_tool import OpenMeteoTool

tool = OpenMeteoTool()
result = tool._run("東京", "2025-11-22")
print(result)
```

## データソース

Open-Meteoは世界中の気象機関のデータを統合しています：
- 日本気象庁（JMA）のMSM/GSMモデル
- ヨーロッパ中期予報センター（ECMWF）
- アメリカ海洋大気庁（NOAA）
- その他多数の国際気象機関

各地域に最適なモデルが自動的に選択されます。

## 料金

完全無料、制限なし。

## トラブルシューティング

### "位置情報が見つかりませんでした"

- 場所の名前を変更してみてください（例：「東京都」→「東京」）
- 英語名で試してみてください（例：「Tokyo」）

### "予報データが見つかりませんでした"

Open-Meteoの予報は7日先までです（設定により最大16日まで拡張可能）。
